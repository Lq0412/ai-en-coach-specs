# ai-design-smell 示例用例

---

## 用例 1：页面评审

**用户输入：**

> 帮我看看这个落地页有没有 AI 设计味。

**期望 agent 行为要点：**

- 按 SKILL 中五大特征逐条体检（蓝紫渐变、卡片套娃、居中堆砌等）
- 每条命中项附证据与具体修复建议
- 总评给出现代简约方向（层级、留白、单主色）

---

## 用例 2：生成时避坑

**用户输入：**

> 生成一个 SaaS 定价页，别有 AI slop，要现代简约。

**期望 agent 行为要点：**

- 生成模式：左对齐、清晰层级、留白充足、单主色无渐变
- 拒绝模板化科技元素（光晕、假 3D、无意义 icon 墙）
- 产出附定稿前自检清单
